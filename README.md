# Clics skills

Agent skills for [Clics](https://clics.dev) — privacy-friendly, cookieless web analytics.

Install with the [skills CLI](https://github.com/vercel-labs/skills):

```bash
npx skills add P-Essonam/clicsdev --skill clics
```

Global install (all projects):

```bash
npx skills add P-Essonam/clicsdev --skill clics -g -y
```

List skills in this repo without installing:

```bash
npx skills add P-Essonam/clicsdev --list
```

## Requirements

- Clics API key from [platform.clics.dev](https://platform.clics.dev)
- MCP: [`@clicsdev/mcp`](https://www.npmjs.com/package/@clicsdev/mcp)
- CLI: [`@clicsdev/cli`](https://www.npmjs.com/package/@clicsdev/cli)

## Skills

| Skill | Description |
|-------|-------------|
| `clics` | Query analytics and manage projects, goals, and funnels via MCP or CLI |
